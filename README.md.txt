
This project is created to practice Python programming for GIS applications.
The goal is to analyze raster-based geospatial data using simple Python scripts.

Tools
- Python
- Rasterio
- NumPy
- Matplotlib


## Progress
- Day 1: Python basics and logic
- Day 2: NumPy arrays and raster-style analysis
- Day 3: CSV handling and attribute-table-style analysis
- Day 4: Pandas for attribute table analysis and filtering
